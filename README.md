🚀 FinBridge AI

AI-powered micro-credit and bank aggregation platform

🧠 Overview

FinBridge AI simulates how modern fintech platforms provide instant loan approvals and financial product recommendations. It uses an explainable AI scoring system to evaluate user risk and aggregate the best loan and FD options from multiple banks.

✨ Features
🔐 Authentication (Email + Google login)
👤 User onboarding (income, age, employment, loans)
🧠 Explainable AI credit scoring (0–100 score with reasons)
💸 Instant loan approval/rejection
🏦 Bank aggregator (SBI, HDFC, ICICI comparison)
🐷 FD calculator (best maturity returns)
📊 Dashboard with score visualization & history
⚙️ Tech Stack
Frontend: React + TypeScript
Backend: Cloud-based (Lovable Cloud / Supabase)
Database: Managed cloud database
AI Logic: Rule-based scoring engine
🔄 How it works
User signs up and completes profile
AI calculates credit score
User applies for loan
System approves/rejects instantly
Shows best bank offers
User compares FD and loan options
🎯 Problem Solved

Access to small loans in India is slow and fragmented.
FinBridge AI provides instant decision-making and unified financial product comparison.
